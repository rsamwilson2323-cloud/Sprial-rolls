# 🌀 Infinity Spiral Trail

An interactive HTML5 Canvas animation that creates glowing infinity-style spiral trails with vibrant rainbow colors, neon lighting effects, and smooth expanding patterns. Designed for both desktop and mobile devices, users can generate mesmerizing spiral animations through mouse and touch interactions.

## ✨ Features

* 🌀 Expanding infinity spiral trails
* 🌈 Dynamic rainbow color transitions
* ✨ Neon glow lighting effects
* 🖱️ Mouse click and drag support
* 📱 Mobile touch interaction
* 🎨 Smooth fading spiral trails
* ⚡ Real-time Canvas rendering
* 🖥️ Fullscreen responsive design
* 🚀 Lightweight and dependency-free

## 🎮 Controls

### Desktop

* Click to create glowing spiral trails.
* Hold and drag to continuously generate expanding spiral effects.

### Mobile

* Tap anywhere to generate infinity spirals.
* Touch and drag to create continuous glowing trail animations.
