# Infinity Spiral Trail ✨🌀

An interactive HTML5 Canvas animation that creates glowing infinity-style spiral trails following mouse or touch movements.

## Features

* Smooth animated spiral effects
* Mouse and touch support
* Dynamic rainbow color transitions
* Neon glow visual effects
* Responsive fullscreen canvas
* Lightweight and dependency-free

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Canvas API

## Usage

1. Open `Sprial rolls.html` in a modern browser.
2. Click, drag, or touch the screen.
3. Watch colorful spirals expand and fade with glowing effects.
